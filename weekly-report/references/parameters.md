# 参数配置

## 必要参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `start_date` | 周报开始日期 | 当前日期 - 4天（含今天共5天） |
| `end_date` | 周报结束日期 | 当前日期 |
| `date_range` | 日期范围类型 | `default`（最近5天）或 `week`（周一到周五） |
| `projects` | 项目路径列表 | 当前工作目录 |
| `format_type` | 周报格式类型 | 按项目分类 |

## 可选参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `output_path` | 周报保存位置 | `桌面/周报/`（如 `C:/Users/用户名/Desktop/周报/`） |
| `extra_work` | 额外工作事项 | 空 |
| `next_week_plan` | 下周工作计划 | 空 |

## 日期格式

所有日期统一使用 `YYYY-MM-DD` 格式：
- 开始日期示例：`2026-05-11`
- 结束日期示例：`2026-05-15`

## 项目路径格式

支持绝对路径、相对路径、多项目逗号分隔，详见 [高级功能 - 路径格式规范](./advanced-features.md#路径格式规范)。

## 格式类型

| 类型 | 说明 |
|------|------|
| `by_type` | 按commit类型分类（功能开发、Bug修复等） |
| `by_project` | 按项目分类（每个项目单独一节） |
